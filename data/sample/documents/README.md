# Синтетический комплект документов для MVP

Этот каталог содержит вымышленные учебные документы. Они созданы только для разработки и тестирования MTR-RAG-System и не должны использоваться для принятия реальных инженерных решений.

Состав комплекта:

- 15 паспортов изделий, по одному на каждую позицию `mtr_catalog.csv`;
- выдержка из условного локального нормативного документа `lnd_extract.md`;
- три текста с намеренно внесенными OCR-ошибками в папке `ocr/`;
- один неполный паспорт для проверки уточняющих вопросов;
- `document_manifest.csv` связывает документ, изделие, код МТР и код КСМ;
- `queries.csv` содержит пользовательские запросы разных режимов;
- `golden_dataset.csv` фиксирует ожидаемое поведение поиска;
- `expected_item_cards.jsonl` содержит эталонные карточки для проверки извлечения.

Разделители `--- Страница N ---` имитируют страницы исходного PDF. После подключения настоящих PDF номер страницы должен приходить от PDF-парсера или OCR-модуля.
